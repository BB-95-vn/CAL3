# Tra Âm -> Dương • Can Chi • Tứ Hóa • Tiết Khí (GitHub Pages)

## Cách dùng
1. Upload toàn bộ folder này lên GitHub repo (giữ nguyên cấu trúc).
2. Bật GitHub Pages: Settings -> Pages -> Branch: main, Folder: /(root)
3. Mở link Pages trên điện thoại là tra được.

## Lưu ý cache
Nếu bạn sửa `app.js`, hãy tăng version ở `index.html`:
`<script src="app.js?v=2"></script>`
